# Mock frontend — Schools Air Quality

Open `index.html` in a browser to view the mock UI. No server required.

Files:
- `index.html` — main UI
- `styles.css` — styles
- `app.js` — sample data and interaction

Usage:
1. Open the file `prototype/frontend-mockup/index.html` in your browser.
2. Click a school on the left to view pollutant RAG ratings and percentages.
